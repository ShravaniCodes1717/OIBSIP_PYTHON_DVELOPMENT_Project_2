BMI Calculator (Python)
Project Overview

The BMI Calculator is a Python-based application that calculates a person's Body Mass Index (BMI) using their weight and height. Based on the BMI value, the program classifies the result into health categories such as Underweight, Normal Weight, Overweight, and Obese.

This project helps beginners understand user input, mathematical calculations, and conditional statements in Python.

Features

Accepts user weight and height

Calculates Body Mass Index (BMI)

Classifies BMI into health categories

Displays the BMI value and category

Simple command-line interface

Installation

Follow these steps to run the project.

1. Clone the Repository
git clone https://github.com/yourusername/bmi-calculator.git

2. Navigate to the Project Folder
cd bmi-calculator

3. Run the Python File
python bmi_calculator.py

Example Input
Enter your weight (kg): 70
Enter your height (m): 1.75

Example Output
Your BMI is: 22.86
Category: Normal Weight
